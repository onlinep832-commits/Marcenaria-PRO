# Programa - Marcenaria-PRO

Sistema de desktop para gestão de módulos e orçamentos de marcenaria.

## Requisitos

- Python 3.6 ou superior

## Como Usar

Execute o programa com:

```bash
python3 marcenaria_pro.py
```

## Funcionalidades

1. **Adicionar Módulo**: Escolha entre vários tipos de módulos pré-definidos
2. **Remover Módulo**: Remova módulos do orçamento
3. **Ver Orçamento**: Visualize o orçamento completo com todos os módulos
4. **Salvar Orçamento**: Salve o orçamento em um arquivo JSON
5. **Variação de Preços**: Adicione variações personalizadas aos preços base
